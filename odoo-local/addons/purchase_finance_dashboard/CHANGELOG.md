# Changelog

Все заметные изменения модуля «Финансовые дашборды».

## 2026-07-23

- Первая версия: `purchase.finance.report` (SQL-view, одна строка на заказ,
  корневой план статьи через нативное `root_plan_id`, заморозка/долг/срочный
  остаток, `is_received` через `stock_picking`).
- Роли `group_ceo`/`group_owner` (+ `is_ceo`/`is_owner` в «Роли сотрудников»,
  `post_init_hook` на известных носителей).
- 4 ролевых дашборда (Собственник/ГД/Финансист/Закупщик), 8 общих экранов
  по затратам, «Без аналитики», `days_to_decide` на согласовании,
  `days_in_stage` (новый механизм отслеживания смены этапа).
- Главный закупщик видит все 4 дашборда, только чтение, без новых прав записи.
- По просьбе пользователя (референс "плитки KPI + графики на одном
  экране") добавлена сводная `spreadsheet.dashboard` страница для
  «Собственника» - приложение «Дашборды» → группа «Закупки» → 4 плитки +
  4 графика, собрана по образцу реального файла Odoo, не вручную.
