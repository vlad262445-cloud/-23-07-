# Changelog

Все заметные изменения модуля «Карточка поставщика».

## 2026-07-24

- Первая версия: финансовые показатели поставщика на лету (выставлено,
  оплачено, едет/заморожено, мы должны, остаток, кол-во заказов, средний
  чек, дата последнего заказа, оборот за период, недостающие документы) -
  без хранения, совпадают с `purchase.finance.report` (модуль 6).
- Карточка `res.partner`: смарт-кнопки «Заявки»/«Платежи»/«Мы должны»,
  вкладки «Финансы»/«Что покупаем»/«Надёжность»/«Реквизиты» - видны только
  поставщикам (`supplier_rank > 0`), доступ ГД/Собственнику - через
  `implied_ids` на `group_observer`, без жёсткой зависимости от
  `purchase_finance_dashboard`.
- Список «Поставщики» (приложение «Реестр закупок»): фильтры по долгу,
  заморозке, отсутствию ИНН, "не заказывали > 90 дней", группировка по
  периоду/стране/городу/полноте карточки.
- Индикатор качества данных (`vendor_data_quality`) и поиск дублей по
  нормализованному ИНН/названию (`vendor_duplicate_group_key`) - экран
  «Найти дубли поставщиков», объединение - штатным мастером Odoo
  (`base.action_partner_merge`), свой визард не писался.
- 10 тестов (совпадение с модулем 6, инварианты по суммам, регрессия на
  `stock.picking` вместо `request_state`, количество запросов не растёт с
  числом поставщиков, поиск дублей по примеру из ТЗ) - 0 failed, 0 error.
