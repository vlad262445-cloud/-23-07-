# Changelog

Все заметные изменения модуля «Архив завершённых закупок».

## 2026-07-22

- Первая версия: статус «Закупка завершена» (`is_completed`).
- Свой флаг `is_archived`/`archived_date` вместо родного `active`.
- Реестр по умолчанию скрывает архивные, с тумблером «Показать архивные».
- Новое меню «Архив закупок».
- Ежедневный cron автоархивации через 5 рабочих дней после завершения.
- Кнопки «В архив» / «Вернуть из архива» на форме заказа.
