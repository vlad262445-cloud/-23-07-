# Changelog

Все заметные изменения модуля «Сопоставление поставщиков по ИНН».

## 2026-07-22

- Первая версия: мастера ИИ-импорта (PDF-счёт, УПД) теперь ищут поставщика
  по нормализованному ИНН (с проверкой контрольной суммы), прежде чем
  сравнивать по названию.
- Примечание в чат заказа/приёмки, если поставщик найден по ИНН, а
  название в документе отличается от названия в карточке.
- ИНН нормализуется (только цифры) при записи — и в новых карточках, и
  при дозаполнении существующих.
- `post_init_hook`: разовая нормализация ИНН у уже существующих
  поставщиков при установке модуля.
