# Changelog

Все заметные изменения модуля «Рабочее место финансиста».

## 2026-07-23

- Первая версия: `finance_stage` (вычисляемый маршрутизатор, 7 значений,
  зафиксированный приоритет при совпадении нескольких условий).
- Приложение «Финансы»: kanban-рабочий стол по этапам + 5 списков
  ("Требуется оплата", "Срочная оплата", "Требуется доплата", "Загрузить
  платёжку", "Сверить ИНН") + "Оплачено за период" (список + pivot).
- `amount_residual_purchase`, `payment_due_date`, `days_waiting_payment`,
  `approval_date`.
